# HVAC Demand Response Thesis Repository

This repository supports the undergraduate thesis project titled "Commercial Building HVAC Dynamic Demand Response".

The repository is for workflow initialization, source organization, decision tracking, and future thesis drafting. It is not a place to store unverified claims, fabricated results, or informal one-off edits without traceability.

## Working Model

- The author makes research and writing decisions.
- Web ChatGPT is used for discussion, critique, review, and alternative phrasing.
- Codex applies confirmed repository changes, maintains structure, and updates tracked project documents.
- GitHub stores version history and supports branch-based review.

## Source-of-Truth Rule

- Markdown is the primary source format for thesis content and project notes.
- Substantive thesis writing should be developed from Markdown sources under `thesis_source/`.
- Word and PDF files are export artifacts for review, submission, or formatting checks.
- Review comments from any source should be reflected back into the Markdown source and notes.

## Repository Areas

- `requirements/`: school requirements, templates, and examples to be uploaded later.
- `style_samples/`: reference theses and papers for writing style study.
- `journal_paper/`: materials related to the IEEE TSTE paper and its assets.
- `research_materials/`: simulation code, data, and research-side working materials.
- `thesis_source/`: future thesis source text in Markdown.
- `figures/` and `tables/`: candidate and selected assets for the thesis.
- `references/`: bibliography and citation planning.
- `notes/`: scope, workflow, decisions, comments, and change tracking.
- `exports/`: generated output packages for review or submission.
- `scripts/`: helper tooling for exports and figure processing.

## Current Stage

This repository currently contains only the initial project structure, workflow notes, and project rules. It does not yet contain thesis chapters, academic claims, final outlines, or validated research conclusions.

# 小论文仿真主线

## Summary

推荐主线是：Stage1/CQR -> RegD 统计 -> 协调备用扫描 -> 固定水侧基线 -> 可信备用筛选 -> 成本/收益后处理 -> xlsx 导出 -> Origin 作图 -> TeX 论文图。

所有主运行入口都集中在 `run/`，所有 xlsx 导出集中在 `export/`。

## Pipeline

| Step | 主程序                                                    | 主要输入                                                     | 主要输出                                                     | 一句话说明                                                   |
| ---- | --------------------------------------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 0    | `startup.m`                                               | 本目录                                                       | MATLAB path                                                  | 加入 `run/export/plot/src`，让统一路径函数可用。             |
| 1    | `run/Stage1Code_CQR.m`                                    | `data/processed/processed_state.mat`, `data/processed/processed_input.mat` | `data/stage1/stage1_cqr_tree_data.mat`, `data/stage1/stage1_hour_model_for_stage2.mat` | 用 15min 状态和输入数据建立 Stage1 小时级模型接口。          |
| 2    | `run/stage1_cqr.py`                                       | `data/stage1/stage1_cqr_tree_data.mat`                       | `data/stage1/stage1_cqr_tree_tuned_parallel_results.mat`, `data/stage1/stage1_cqr_for_stage2.mat` | 训练 CQR 树模型并生成 Stage2 需要的预测区间结果。            |
| 3    | `run/stage1_cqr_plot.py`                                  | Stage1 CQR MAT                                               | `data/figures/replot_*.png`                                  | 复查 Stage1 覆盖率、区间宽度和代表日预测区间。               |
| 4    | `run/run_regd_15min_historical_variance_v8.m`             | `data/raw/07 2020.xlsx`                                      | `data/regd/regd_15min_moments_and_hourly_cov_dynamic.mat`    | 从 RegD 历史信号计算 15min 均值、方差和小时内协方差。        |
| 15   | `run/run_ch3_hourly_reserve_costcurve_main_V2.m`          | Stage1 模型/CQR MAT                                          | `data/reserve/hourly_reserve_costcurve_all_beta.mat`         | 逐小时扫描备用 R、供水温度 Ts、风量 ma，得到调度可行备用和成本曲线。 |
| 6    | `run/run_ch3_baseline_fixedTs_suite_V4.m`                 | Stage1 模型/CQR MAT                                          | `data/baseline/baseline_fixedTs_multi_*_v4.mat`              | 生成固定水侧 Ts 的 air-only 基线，用于与水风协调策略对比。   |
| 7    | `run/run_ch3_hourly_credible_feasibility_screen_V2.m`     | `data/reserve/hourly_reserve_costcurve_all_beta.mat`, RegD 统计 MAT | `data/reserve/hourly_credible_feasibility_v2_all_beta.mat`   | 结合调频扰动和风险边界，从可行备用中筛出可信备用。           |
| 8    | `run/run_postprocess_hourly_credible_cost_view_full.m`    | 备用扫描 MAT、可信筛选 MAT                                   | `data/postprocess/cost/hourly_reserve_credible_cost_full_beta_95.mat` | 生成可信备用域内的纯成本后处理结果。                         |
| 9    | `run/run_postprocess_hourly_credible_revenue_view_full.m` | 备用扫描 MAT、可信筛选 MAT                                   | `data/postprocess/market/hourly_reserve_credible_revenue_full_*_beta_*.mat` | 生成包含容量收益或 no-revenue 版本的市场视角后处理结果。     |
| 10   | `export/*.m`                                              | 上述 MAT                                                     | `data/exports/<按图号分组>/*.xlsx`                           | 把论文和 Origin 需要的数据导出为 xlsx。                      |
| 11   | `origin/figure1.opju`                                     | `data/exports/<按图号分组>/*.xlsx`                           | `paper/v3_TSTE.pdf` 中的图                                   | Origin 读取导出表作图，最终图被 TeX 引用。                   |

# 图表素材目录说明

本目录用于管理本科毕业论文图表素材、重画中间文件和最终入文图。

## 目录结构

- `source/`
  - 保存原始图表素材或重画参考素材。
- `source/tste/`
  - 保存 IEEE TSTE 小论文图表素材副本。
- `source/midterm/`
  - 保存中期阶段已绘制或可能用于重画参考的图表素材。
- `working/`
  - 保存中文化、重画、合并、排版过程中的中间图源。
- `final/`
  - 保存最终准备插入本科论文正文的中文图。

## 使用边界

- 当前导入的图表均为候选素材，不代表最终入文。
- 图表最终是否入文、最终图号、图题和正文解释，以后续章节写作和作者确认为准。
- `journal_paper/` 与中期原始材料仍分别是小论文图和阶段性研究图的原始参考来源。

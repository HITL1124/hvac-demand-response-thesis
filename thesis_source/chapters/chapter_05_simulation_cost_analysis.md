# 第5章 仿真结果与可信备用成本分析

## 1. 本章功能

- 作为本科论文的仿真结果和分析章。
- 说明仿真算例、数据来源、对比方案和评价指标。
- 分析水-空气协调工作点重构效果。
- 分析最大可行备用与可信备用结果。
- 单独分析可信备用能力成本曲线。
- 成本口径保持待确认。

## 2. 本章主线

```text
仿真算例与评价设置
→ 水-空气协调工作点重构效果
→ 最大可行备用与可信备用对比
→ 可信备用能力成本曲线
→ 回扣动态需求响应能力评估
```

## 3. 章节小节设计

- `5.1 仿真算例与评价设置`
- `5.2 水-空气协调工作点重构效果分析`
- `5.3 最大可行备用与可信备用结果分析`
- `5.4 可信备用能力成本曲线分析`

## 4. 各小节拟写内容

### 5.1 仿真算例与评价设置

- 拟说明仿真算例、数据来源和时间尺度。
- 拟说明对比方案，例如协调策略、固定水侧或 air-only 基线。
- 拟说明评价指标，但不编造数值。
- 拟交代图表来源应来自已确认的仿真材料和图表登记。

### 5.2 水-空气协调工作点重构效果分析

- 拟分析工作点重构对供水温度、风量、室温和功率构成的影响。
- 拟说明协调工作点重构如何体现水-空气耦合机理。
- 可对应 TSTE Fig6、Fig7、Fig10 等候选图。
- 不重复第4章模型推导。

### 5.3 最大可行备用与可信备用结果分析

- 拟对比天然备用、最大可行备用和可信备用。
- 拟说明可信备用小于或等于最大可行备用的风险约束原因。
- 可对应 TSTE Fig9、Fig11、Fig14 等候选图。
- 不写未核对的具体数值或最终图号。

### 5.4 可信备用能力成本曲线分析

- 拟单独分析可信备用能力与成本之间的关系。
- 可对应 TSTE Fig12、Fig13 等候选图。
- 成本口径保持待确认。
- 不写死 holding cost、net cost、noRevenue 或 revenue。

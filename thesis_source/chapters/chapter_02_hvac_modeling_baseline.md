# 第2章 商业建筑 HVAC 系统建模与风机功率基线形成机理

## 1. 本章功能

- 说明本文研究对象、系统边界和 HVAC 功率构成。
- 建立水-空气耦合换热、风机功率和热源侧功率的基础表达。
- 解释供水温度影响风机功率基线的机理。
- 介绍 Dymola 高保真模型和数据基础。
- 明确 Dymola 是模型和数据基础，不是论文唯一核心方法。

## 2. 本章主线

```text
HVAC 系统结构与建模边界
→ 水-空气耦合换热与功率模型
→ 供水温度影响风机功率基线的机理

→ 为第4章工作点重构提供物理基础
```

## 4. 各小节拟写内容

### 2.1 商业建筑 HVAC 系统结构与建模边界

- 先说明 HVAC 系统结构和本文建模边界。
- 拟交代建筑热区、空气侧、盘管、水侧和热源侧之间的关系。
- 不在本节一开始写 Dymola。
- 不展开第4章的工作点重构优化。

### 2.2 水-空气耦合换热与 HVAC 功率模型

- 拟说明盘管换热关系、空气侧热容量率和水侧/空气侧耦合关系。
- 拟说明风量变化与风机功率之间的关系。
- 拟说明热源侧功率和 HVAC 总功率分解。
- 拟为后续备用能力评估提供功率模型基础。

### 2.3 水-空气耦合下的风机功率基线形成机理

- 拟解释为什么供水温度变化会改变盘管换热能力。
- 拟解释盘管换热能力变化如何影响空气侧风量需求。
- 拟解释风量变化如何改变风机功率基线。
- 只写机理，不写面向目标备用水平的工作点重构优化。

### 2.4 Dymola 高保真模型与数据基础

- 拟介绍 Dymola 高保真模型的来源、系统组成和数据基础。
- 拟说明 Dymola 与 Modelica/Buildings Library 的关系。
- 拟说明 Dymola 材料用于支撑模型对象、数据来源和仿真基础。
- 不把 Dymola 写成本文唯一核心方法或独立贡献点。

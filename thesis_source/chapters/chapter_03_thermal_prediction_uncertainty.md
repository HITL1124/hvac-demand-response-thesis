# 第3章 面向动态需求响应的建筑热动态预测与不确定性表征

## 1. 本章功能

- 建立面向动态需求响应分析的建筑热动态预测模型说明。
- 说明 N4SID 在获得状态空间热动态模型中的作用。
- 说明 CQR 如何用于温度不确定性表征。
- 构建风险收紧舒适约束，作为第4章可信备用筛选的热可行性接口。
- 本章不直接筛选可信备用能力。

## 2. 本章主线

```text
建筑热动态预测需求
→ 状态空间热动态模型
→ N4SID 用于模型辨识
→ CQR 表征温度预测不确定性
→ 风险收紧舒适约束
→ 提供给第4章可信备用筛选使用
```

# 第1章 绪论

## 1. 本章功能

- 提出本科毕业论文的研究问题和写作背景。
- 从新型电力系统灵活调节需求引出商业建筑 HVAC 系统动态需求响应。
- 说明本文将“备用能力”作为量化 HVAC 动态需求响应能力的一种方式。
- 交代本文研究内容和章节安排。
- 本章不设置“本章小结”。

## 2. 本章主线

```text
新型电力系统灵活调节需求
→ 需求响应与柔性负荷资源
→ 商业建筑 HVAC 系统动态响应潜力
→ 备用能力作为动态需求响应能力的量化表达
→ 本文研究内容与章节安排
```

## 3. 章节小节设计

- `1.1 研究背景与意义`
- `1.2 国内外研究现状`
- `1.3 本文主要研究内容与章节安排`

## 4. 各小节拟写内容

### 1.1 研究背景与意义

- 拟说明新型电力系统对灵活调节资源的需求。
- 拟说明商业建筑 HVAC 系统具有参与需求响应的负荷基础和热惯性基础。
- 拟说明动态需求响应不能只看功率变化，还需要考虑舒适约束、设备状态和可交付能力。
- 拟引出备用能力、最大可行备用能力和可信备用能力作为本文后续分析对象。

### 1.2 国内外研究现状

- 可按三级标题组织，不在本模板中写正文。
- 建议方向：
  - `1.2.1 建筑 HVAC 动态需求响应研究`
  - `1.2.2 HVAC 备用能力表征研究`
  - `1.2.3 不确定性与可信调节能力评估研究`
- 每类综述后需要归纳其与本文研究问题的关系。
- 避免逐篇罗列文献。

### 1.3 本文主要研究内容与章节安排

- 拟概括本文围绕商业建筑 HVAC 动态需求响应和备用能力评估展开。
- 拟说明第2章到第5章的功能分工。
- 章节安排只说明功能，不写成最终贡献点。

# 第2章 商业建筑 HVAC 系统数学建模与高保真仿真基础

## 1. 章节标题

第2章 商业建筑 HVAC 系统数学建模与高保真仿真基础

## 2. 本章功能

- 建立商业建筑 HVAC 系统对象、数学模型和 Dymola 高保真仿真基础。
- 说明系统运行机理、能量关系、功率构成和运行约束。
- 为后续热动态预测、备用能力评估和仿真结果分析提供对象基础与数据基础。

说明：

- 第2章只讲系统机理、数学模型和仿真基础。
- 本章不提前讨论备用能力、最大可行备用能力、可信备用能力、风机功率基线形成或工作点重构。

## 3. 本章主线

```text
→ 水侧、空气侧和建筑热区的能量关系

```

# 第4章 基于工作点重构与部署可信筛选的 HVAC 备用能力评估方法

## 1. 本章功能

- 建立 HVAC 备用能力评估方法框架。
- 区分备用能力、最大可行备用能力和可信备用能力。
- 说明工作点重构主要对应最大可行备用能力。
- 说明部署风险约束筛选才对应可信备用能力。
- 调用第3章风险收紧舒适约束，形成可信备用筛选条件。

## 2. 本章主线

```text
备用能力层级关系
→ 面向目标备用水平的工作点重构
→ 得到调度可行/最大可行备用能力
→ 调用风险收紧舒适约束
→ 部署可信筛选
→ 得到可信备用能力
```

## 3. 章节小节设计

- `4.1 HVAC 备用能力层级与评估框架`
- `4.2 面向目标备用水平的工作点重构模型`
- `4.3 基于部署风险约束的可信备用筛选`

## 4. 各小节拟写内容

### 4.1 HVAC 备用能力层级与评估框架

- 拟定义或说明备用能力、最大可行备用能力和可信备用能力的层级关系。
- 拟说明最大可行备用能力与可信备用能力的区别。
- 拟说明本章评估框架如何承接第2章模型和第3章风险约束。
- 不把层级关系写成最终图号。

### 4.2 面向目标备用水平的工作点重构模型

- 拟说明给定目标备用水平后，如何通过供水温度、风量等变量重构水-空气协调工作点。
- 拟说明工作点重构如何得到调度可行或最大可行备用能力。
- 不重复第2章供水温度影响风机功率基线的机理说明。
- 不把最大可行备用能力写成可信备用能力。

### 4.3 基于部署风险约束的可信备用筛选

- 拟说明备用部署后需要考虑室温风险、舒适约束和预测不确定性。
- 拟调用第3章风险收紧舒适约束。
- 拟说明部署风险约束筛选如何从最大可行备用中得到可信备用能力。
- 不把可信备用筛选与工作点重构混写。
